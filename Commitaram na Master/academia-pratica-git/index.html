Git e Github. Para que serve e como usar?

Por: Pablo Jorge

Git :

Git é um software de versionamento, que permite o acesso a versões antigas do projeto. A principal característica é que podemos criar pontos no repositório local que salva o estado atual do projeto e caso necessário podemos voltar e recuperar o estado salvo do projeto naquele ponto.

Github:

Github é um repositório remoto, que permite a hospedagem de projetos git. O mesmo facilita o trabalho em equipe, pois através dele é possível fazer a junção de partes do projeto impedindo que um sobrescreva o outro. Uma outra funcionalidade é o acesso a repositórios de outros usuários, podendo clona-los e até contribuir com os projetos.

Principais Comandos Git

git init (inicializa um repositório);
git status(status do repositório);
git add (Adiciona o conteúdo do arquivo ao índice);
git commit -m"mensagem"(salva o estado atual do projeto no repositório local);
git pull origin nome-do-branch (traz histórico do servidor remoto);
git push origin nome-do-branch(envia histórico ao servidor remoto);
git branch (lista as ramificações);
git branch nome-do-branch (cria uma ramificação);
git checkout nome-do-branch (troca de ramificação);
git merge nome-do-branch (faz a junção de uma ramificação "filha" a branch "mãe/master");
